Joonchicken.github.io

Winter 2024-25, Summer 2025
Last updated 7/17/25


This was my first large project, and my first time making a website from scratch with HTML, CSS, and JS. The last time I even attempted web development was in middle school, when I took a really tedious HTML/CSS basics summer program at a local high school. I had to relearn everything I'd forgotten (so, basically, I had to relearn everything), but I also got to learn cool new stuff like making procedural textures with JS and <canvas>, and the basics of JQuery and React. Once I understood the difference between margin and padding and what box-sizing: border-box meant, the design process flowed smoothly, and when I realized how powerful JQuery animations and the JQuery UI library were, the ideas flowed too (and so did the feature creep).

As this project was always meant as more of a hands-on learning experience rather than a beautifully polished final product, and because I focused a lot more on design in the early phases (since I come from a more artistic background and it's just what I'm more comfortable with at the moment) there's a ton of jank lying around in the code, including but not limited to*:

  • JQuery fails to load sometimes (idk how to fix it I've tried so hard I swear)
  • Horrendous vestigial CSS (something will break if I refactor it)
  • Uhh... am I allowed to make JQuery calls inside a React application???
  • Indecision regarding document.getElementById("id") vs. $("#id")
  • Am I doing this right?????
  • And much more!
So enjoy the jank while you can, until I try to make changes in five years and break everything.

Thanks to my friend Henry Ty <Website / Github> for introducing me to the beautifully chaotic world of JavaScript because this website probably would have been static and boring otherwise.

Thanks also to Scrimba for their free course on React basics! The project manager wouldn't have been possible without this couse. Their style of interactive teaching was so great for quickly getting acquainted to React and its quirks.
