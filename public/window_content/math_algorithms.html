Programs

  • Darcy.py

    Using Darcy's Law, this program simulates a fluid flowing through consecutive sections of porous media in a single tube with a uniform cross section and outputs the steady state pressure at each section.
    Pressure may or may not be uniform throughout each section, but I don't think it is so I've kept the output as a line graph.

    Params:

    area .............. cross-sectional area of the tube
segmentLengths .... array w/ lengths of each segment
k ................. array w/ permeability of each segment
mu ................ array w/ dynamic viscosity of fluid in
                    each segment
p_0 ............... pressure constraint on left end of tube
p_n ............... pressure constraint on right end of tube

  • Heat.py

    This program simulates the transfer of heat along a conductive rod by numerically solving the Heat equation with the forward finite difference method and outputs an animated graph of heat transfer in action over the length of the rod.

    Params:

    length ............ length of the rod
k ................. conductivity of the rod
temp_left ......... temperature constraint on left end of rod
temp_right ........ temperature constraint on right end of rod
total_time ........ number of time steps to run the simulation
                    for 
dx = 0.9 .......... size of subdivision of the rod
dt = 0.19 ......... size of time step

  • LU_Decomp.py

    This program presents a computationally efficient method of calculating the determinant of a matrix by first factoring it into two triangular matricies using Lower-Upper Decomposition (more specifically LU factorization with partial pivoting), then multiplying together all the elements in the main diagonal of the resulting upper triangular matrix to calculate the determinant of the original matrix.

    Params: accepted from console input

    number of rows
number of columns
elements in the matrix

  • RK.py

    This program efficiently and accurately approximates a given differential equation with a Runge-Kutta method specified by a Butcher Tableau. A sample function and its derivative are provided for easy comparison.

    Params:

    xi ................ x initial; where the algorithm starts the
                    approximation
xf ................ x final; where the algorithm ends the
                    approximation
yi ................ y initial; since diffeqs describe a family
                    of functions, this y-value chooses one
dx ................ size of step in x
Tab ............... the Butcher tableau to use in this
                    approximation (includes RK4, RK3, RK38,
                    Euler, ImplicitEuler)
